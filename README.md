# MesaFacil-2026

Aplicação para agendamento de reservas em um restaurante.

## Regras de Negócio
- Requisitos de informação do usuário para fazer o agendamento
- Reservas só podem ser feitas com antecedência mínima de 3 horas
- Cancelamento permitido até 1 hora antes do horário marcado
- Limite de 15 pessoas por reserva
- Limite total de pessoas no restaurante por dia para não sobrecarregar
- Status da reserva: pendente, confirmada, cancelada
